简易版介绍：实际生产环境中并不推荐使用外键，因为级联风暴之类的等等麻烦。

所以，本项目旨在提供应用层面的外键，并通过图数据库提供不同表字段直接的关系展示，以及如何快速简洁的从一个表的某字段到另一个表的某字段，为未捋清表关系新人以及懒得动脑思考表关系的老人提供crud赋能
